# CSV Uploader Frontend

A React frontend for uploading and downloading CSV files processed asynchronously via a gRPC backend.

## Features

- Upload CSV files to the backend for processing
- Monitor processing status and progress
- Download processed CSV results
- Real-time updates on lines processed, departments, and time elapsed

## Installation

```bash
npm install
```

## Usage

Start the development server:

```bash
npm run dev
```

The frontend will be available at http://localhost:5173.

Ensure the backend services are running as described in the root README.md.
